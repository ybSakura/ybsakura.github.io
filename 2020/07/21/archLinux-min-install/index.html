安装基础Arch系统

  1. 进入archiso进行安装
  2. 使用wget下载archfi脚本(作者已取消支持)
  • 注：archlinux在6月后的版本取消了对wget打支持,使用手机usb网络共享时会自动配置usb网络
  • 如需要使用wget可以使用pacman -Sy wget安装wget

3.使用curl下载 archfi脚本 curl -L archfi.sf.net/archfi > archfi或curl -L matmoul.github.io/archfi > archfi
4.使用 sh archfi 启动
